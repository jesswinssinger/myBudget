# README

# myBudget
Simple Ruby on Rails budgeting app project

## Design
### Interface
* Create expense categories
  * Defaults: Home, Hygiene, Eating Out
* Create income categories
  * Default: Work
    ** Set salary or hourly wage
    ** Repeats - auto-log at a specific date
* Add an expense
  * Date (default today)
  * Category
    ** Option to add new category
  * Item
  * Store/Vendor
  * Amount
* Add income
  * Date
  * Amount
* savings goal
* shopping list

### Infrastructure
* Database
  * Users
  * Expense Categories
  * Income Categories
  * Expenses
